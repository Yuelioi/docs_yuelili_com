#  getbounces

VEX function  
#


```c
int  getbounces(bsdf b)
```


Returns the bounce mask for the specified `bsdf`.

返回指定bsdf的反弹掩码。

See [bouncemask](bouncemask.html) for information on component label bitmasks.

关于组件标签位掩码的信息，请参见bouncemask。
