Returns 1 if the Vex CHOP‘sUnit Menu is currently set to 'seconds',  
0 otherwise.



Returns 1 if the Vex CHOP‘sUnit Menu is currently set to 'seconds',
0otherwise.

如果Vex CHOP's Unit Menu当前设置为 "秒"，则返回1，否则返回0。
