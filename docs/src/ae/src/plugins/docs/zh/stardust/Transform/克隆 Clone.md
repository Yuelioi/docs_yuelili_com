## 克隆

创建一个新的粒子系统。与 ctrd d 有点像，不过更便于调节。

### 使用方法

将“克隆”节点连接到其他节点，这能复制一个新的粒子系统。

一般使用 Transform 来移动克隆过的分支，并重新设定发射器的种子值（random seed）。

### 参数

Delay（Seconds） 延时（秒）：延时原始粒子的系统

Shift Seed 种子变换：改变种子值
