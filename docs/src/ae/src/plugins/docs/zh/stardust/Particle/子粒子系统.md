## 子粒子 Aux

主发射器发射主粒子，再把发出的主粒子作为发射器。

子粒子系统和＂发射器＂章节完全一致，不再赘述（除了 Auxiliary 属性外），因此其他属性详细介绍请参考发射器

设置辅助后，其父发射器将用作源位置，并且可以继承一些属性，例如寿命，颜色，大小等。
