## 参数详解

滴落速度：每秒的波纹数量

持续时间：

波纹：波纹的圈数

置换：与周围环境（原始图像）进行置换。或者说是扭曲 - 把船给扭了

![](https://cdn.yuelili.com/20211212155836.png)

波纹高度：波纹宽度

扩散：波纹大小
