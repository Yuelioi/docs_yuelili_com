## 简述

把 RGBA 通道换成其他层的通道。如 Alpha 及 RGB、亮度、饱和度等通道。

例如，不同层的图像的亮度值可以用于所选层的 Alpha。

例如，您可以获取控件图层像素的明亮度值，并将其用作效果图层像素的蓝色通道值。

此效果适用于 8-bpc 和 16-bpc 颜色。
