## 简述

**旧版本** 使用此“路径文本”效果；否则，应使用 **文本图层** 来更好地控制文本格式和文本动画

“路径文本”效果使您能够对路径上的文本设置动画。您可以将路径定义为一条直线、任意直径的圆或者贝塞尔曲线。您也可以导入在其他应用程序（例如 Adobe
Photoshop 或 Adobe Illustrator）中创建的路径。“路径文本”效果可以与非方形像素一起使用，并相应地调整字符形状和路径形状。

此效果适用于 8-bpc 颜色。
