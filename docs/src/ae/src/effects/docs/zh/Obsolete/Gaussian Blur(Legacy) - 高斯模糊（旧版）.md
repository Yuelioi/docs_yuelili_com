## 简述

高斯模糊效果可使图像变模糊，柔化图像并消除杂色。图层的品质设置不会影响高斯模糊效果。

与高斯模糊效果相比，高斯模糊（旧版）效果所产生的渲染结果稍有不同。

此效果适用于 8-bpc、16-bpc 和 32-bpc 颜色。
