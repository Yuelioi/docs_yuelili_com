## 简述

将位于指定半径之内的不同像素更改为更类似邻近的像素。

此效果适用于 8-bpc 和 16-bpc 颜色。

## 参数详解

### 半径

该效果搜索像素间差异的距离。较高的值会使图像模糊。请使用能够消除瑕疵的最小值。

### 阈值

像素能够与其邻近像素在多大程度上不同而不被效果更改。使用能够消除瑕疵的最高值。
