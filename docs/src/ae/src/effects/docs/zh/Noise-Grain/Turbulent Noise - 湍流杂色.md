## 简述

湍流杂色跟[分形杂色](https://www.yuelili.com/docs/ae-effect/fractal-noise/)效果完全一样。区别在于湍流杂色渲染更快，但是没有循环控件。

如果要循环就分形杂色，不然湍流杂色。
