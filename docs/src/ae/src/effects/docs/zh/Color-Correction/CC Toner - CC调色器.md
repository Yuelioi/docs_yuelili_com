## 简述

相当于填充、色调、三色调的结合体。用于颜色映射。分别映射衍射到高光、明亮、中间调、暗色调、阴影。中间则按比例映射

## 参数详解

色调：设置颜色模式

高光：映射到高光的颜色

明亮：映射到明亮的颜色

中间调：映射到中间调的颜色

暗色调：映射到暗色调的颜色

阴影：映射到阴影的颜色

原始混合：与原始图像混合程度。100 则完全没效果
