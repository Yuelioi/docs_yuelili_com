- 使用颜色键等提取颜色时，您可以 **填充图像内部** 的 **孔** 。
- 与 [简单阻塞] 效果不同，您可以同时使用一种效果同时放大/缩小裁剪范围。

## 参数详解

### 几何柔和

可以调整效果引起的变化范围，数字越大，效果范围越广。

### 阻塞

可以调整切出的范围，数字越大，范围越小，数字越负，范围越大。

### 灰阶柔和

可以调整边缘的处理，数值越大，边缘越模糊。

### 技巧

一般分两个阶段进行。  
第一阶段是用来填补被切掉的孔洞，第二阶段是用来恢复边缘，以保持图像的形状。
