## 参数详解

### 调制类型

要使用的波形的类型。正弦可产生更平滑的调制效果。三角形可产生更突然的调制效果。

### 调制速率

调制的速率，以赫兹为单位。

### 调制深度

调频量。

### 振幅变调

振幅变调量。
