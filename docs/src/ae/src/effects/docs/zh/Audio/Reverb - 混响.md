## 参数详解

### 混响时间

原始音频和混响音频之间的平均时间，以毫秒为单位。

### 扩散

指定效果散布原始音频的量。扩散得越多，音频声音听起来距离麦克风越远。

### 衰减

指定效果衰减需要的时间。值越大，模拟的空间效果越大。

### 亮度

指定留存的原始音频中的细节量。亮度值越大，模拟的室内反射声音效果越大。

### 干输出、湿输出

最终输出中的原始（干）声音量和延迟（湿）声音量。值 50% 是常用值。
