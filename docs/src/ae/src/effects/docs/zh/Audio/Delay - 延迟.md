## 简述

延迟效果可在指定时间后重复音频。此效果可模拟从某表面（如墙壁）弹回的声音。

要模拟房间的声音环境，请使用混响效果。

## 参数详解

### 延迟时间

原始声音及回音之间的间隔，单位为毫秒。

### 延迟量

作为原始声音一部分的第一个延迟音频量。

### 反馈

为创建后续回音反馈到延迟线的回音量。

### 干输出、湿输出

最终输出中的原始（干）声音量和延迟（湿）声音量。值 50% 是常用值。
