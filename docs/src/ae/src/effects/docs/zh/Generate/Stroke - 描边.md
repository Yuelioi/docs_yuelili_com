## 简述

根据一个或多个蒙版定义的路径周围创建描边或边界。

相当于高级版的图层样式 - 描边，多了裁剪、羽化、间距

或者低级版的形状图层描边，少了虚线、波形、间距样式

要使用在 Illustrator 中创建的路径，请复制此路径，并将它粘贴至 After Effects 中的图层。

此效果适用于 8-bpc 颜色。
