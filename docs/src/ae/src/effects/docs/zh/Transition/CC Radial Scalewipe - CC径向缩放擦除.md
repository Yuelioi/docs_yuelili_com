## 简述

从中心向四周擦除，呈圆状。过渡边界的像素会扭曲，从而与边界交融。

## 效果展示

过渡是完全擦掉，0 不透明度！但是为了演示，底部放了一个 30%不透明度的原图。

中心：过渡的中心点
