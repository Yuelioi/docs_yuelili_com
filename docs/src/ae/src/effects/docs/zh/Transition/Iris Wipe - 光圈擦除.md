## 简述

光圈擦除效果基于多边形进行擦除。其实就相当于创建一个多边形形状的蒙版！只不过点数/2 = 多边形的边数

光圈擦除效果是唯一不具有过渡完成属性的过渡效果。使用“半径”属性设置动画。

此效果适用于 8-bpc 和 16-bpc 颜色。在 After Effects CS6 或更高版本中，此效果在 32 位颜色中有效。
