## 简述

百叶窗效果使用具有指定方向和宽度的条显示底层图层。

使用“草图”品质时，这些条以像素精度设置动画；使用“最佳”品质时，这些条以子像素精度设置动画。

此效果适用于 8-bpc 和 16-bpc 颜色。
