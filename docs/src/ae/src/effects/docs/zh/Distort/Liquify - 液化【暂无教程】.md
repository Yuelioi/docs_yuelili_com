## 参数详解

### 工具

选择变形的工具

- 湍流  
  拖动区域不规则扭曲

- Shrink  
  向指针中心收缩

- 像素移动  
  拖动部分扭曲成直角

- 重建 恢复  
  已经扭曲的部分

![](https://cdn.yuelili.com/20211224181556.png)

### 扭曲网格

点击关键帧来为扭曲设置动画。

### 失真率

可以指定失真的强度，  
数值越大失真越大，数值越小越接近原图。
