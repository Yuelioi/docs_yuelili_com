## 简述

可以扭曲图层，就像是用镜头拍摄的一样。使用类似“光学补偿”动画进行屏幕转换。

居然不能对蒙版生效。

感觉对分形杂色的相性比极坐标好？

### Convergence 收敛

镜头的不平整程度。负值是向前方突出的镜头，而正值是凹陷的镜头。

![](https://cdn.yuelili.com/20211222170956.png)
