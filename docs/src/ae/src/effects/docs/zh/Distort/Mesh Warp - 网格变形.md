## 简述

通过自定义的网格点，进行扭曲相关区域。进阶版的贝塞尔变形

### 关键词

效果：

类型：扭曲

控制基于：

控制范围：点控制、区域控制

生效范围：不透明度、位置、扭曲

## 参数详解

### 行数与列数

最多可以设置 31 行和 31 列的网格。数值越大，图像控制越精细。

![](https://cdn.yuelili.com/20211225012434.png)

### 扭曲网格

通过单击秒表并点击关键帧来创建变形动画。
