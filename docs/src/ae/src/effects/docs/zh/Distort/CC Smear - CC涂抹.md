## 参数详解

### From / To

决定要被扭曲的区域  
畸变发生在从 From 到 To 的一条直线上

### Reach

扭曲的距离

![](https://cdn.yuelili.com/20211223003426.png)

### Radius

扭曲的范围，可以缩放

![](https://cdn.yuelili.com/20211223003650.png)
