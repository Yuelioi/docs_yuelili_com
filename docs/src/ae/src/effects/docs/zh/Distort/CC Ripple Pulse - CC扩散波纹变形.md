### 特点

## 参数详解

### Pulse Level (Animate) 脉冲水平

可以通过更改两个不同的值作为关键帧来制作动画  
两个值之间的差异越大，波浪就越大。

![](https://cdn.yuelili.com/20211222181036.gif)

### Time Span (sec) 时间延展

决定波浪将保持多少秒。

### Amplitude 振幅

波浪的扭曲程度

![](https://cdn.yuelili.com/20211222181331.png)

### Render Bump Map(RGBA)

如果选中，扭曲的区域将以黑白图像显示。

![](https://cdn.yuelili.com/20211222181539.png)
