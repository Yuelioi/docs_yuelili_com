## 参数详解

### Blobbiness

Blob Layer

要参考的层

Property 属性

Blob Layer 层要引用的信息。

Softness 柔软度

允许你调整图层的柔软度。柔软度越低，越接近颗粒状、线条状

### Light

Using 使用

使用效果灯，只适用于这个效果的灯光，或者使用 AE 灯，也就是合成的灯光

Light Type 灯光类型

远光，就像阳光照射在整个图像上，

点光。只有在使用效果灯时才能调整。

### Shading 阴影

用环境和漫反射调整整个物体的亮度和暗度  
用 Specular、Roughness 和 Metal 调整压花区域的纹理。
