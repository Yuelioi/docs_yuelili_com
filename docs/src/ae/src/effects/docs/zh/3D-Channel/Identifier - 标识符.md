## 简述

可以使用作为 HDI 文件的一种的 Openexr 文件中包含的对象 ID 或素材 ID，制作每个 ID 的蒙版。

## 参数详解

### Channel Info 通道信息

选择要使用的项目 ID 的种类。

### Display 播放

所选择的 ID 的显示方法。有[Colors]、[Luma Matte]、[Alpha Matte]和[Raw]。

### ID

设定要显示的 ID
