## 简述

将灰度或彩色图像转换为高对比度的黑白图像。

指定特定的级别作为阈值（分界线）；比阈值浅的像素转换为白色，比阈值深的像素转换为黑色。

此效果适用于 8-bpc 和 16-bpc 颜色。
