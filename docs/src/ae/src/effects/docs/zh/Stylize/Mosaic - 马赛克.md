## 简述

马赛克效果可使用纯色矩形填充图层，以使原始图像像素化。

可用于模拟低分辨率显示，以及遮蔽面部。也可以用来做过渡动画。使用“最佳”品质时，矩形边缘会消除锯齿。

此效果适用于 8-bpc 和 16-bpc 颜色。

## 参数详解

### 水平/垂直块

像素块的个数，如果跟原图像一样大，则不会像素化

### 锐化颜色

对结果进行锐化
