## 简述

彩色浮雕效果的作用与浮雕效果一样，但不会抑制图像的原始颜色。

- 增加对比度
- 增加深度

此效果适用于 8-bpc 和 16-bpc 颜色。

## 参数详解

### 方向

高光源发光的方向。

### 起伏

浮雕的高度，以像素为单位。“起伏”设置实际用于控制高光边缘的最大宽度。

### 对比度

图像的锐度。

### 与原始图像混合

效果的影响程度，是效果图像与原始图像混合的结果。值越高，效果影响越小。 100%，则几乎没影响。
