## 简述

在图形四周形成暗角。有点像聚光灯打在图像上，未打到的区域会变暗。

## 参数详解

### 数量

“聚光灯”的强度

### 视角

“聚光灯”的夹角

### 中心

位置

### 中心高光

值越大，高光区受到影响越小
