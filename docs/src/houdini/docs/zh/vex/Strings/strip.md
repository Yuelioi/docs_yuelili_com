`string strip(string value)`

`string strip(string value, string whitespace)`

返回一个去除前导和尾部空白的字符串。如果提供了空白，它是一个将被剥离的字符串。

这也相当于在一个字符串上做[rstrip](rstrip.html) ()（"去除字符串的尾部空格"）和[lstrip](lstrip.html)（"去除字符串的头部空格"）。
