`bsdf blinn(float exponent, ...)`

`bsdf blinn(vector nml, float exponent, ...)`

返回一个 Blinn BSDF。

关于 BSDF 的信息，请参见[编写 PBR 着色器](../pbr.html)。

`vector blinn(vector nml, vector V, float roughness, ...)`

计算 Blinn 的阴影。

## Arguments

`nml`

用于评估的表面的法线。

`V`

发生率向量。

`exponent`

指数值。该值越高，镜面层越紧密。

每个功能还可以采取一个可选的光罩。
