[chop](../contexts/chop.html)

`float chstartt()`

上下文 使用`-1`作为`opinput`。

`float chstartt(int opinput)`

返回与给定的 CHOP 输入的第一个样本对应的时间（以秒为单位）。

要获得起始样本号，请使用 [chstart](chstart.html) () ("返回指定输入的起始样本。")。要获得起始样本框号，使用[chstartf](chstartf.html) ("返回指定输入的第一个样本对应的框。")。
