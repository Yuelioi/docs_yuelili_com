[chop](../contexts/chop.html)

`int chindex(int opinput, string name)`

`int chindex(string name)`

`int [] chindex(string names[])`

Context(s) 从一个给定的通道名称的输入中返回通道索引，或者在失败时返回-1。

## Arguments

`opinput`

CHOP 输入索引，如果省略则为-1。

`name`

要查询的频道名称。

`names`

要查询的通道名称的数组。
