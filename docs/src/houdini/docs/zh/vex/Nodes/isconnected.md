`int isconnected(int opinput)`

## Arguments

`opinput`

Context(s) 要读取的输入号码，从 0 开始。例如，第一个输入是 0，第二个输入是 1，以此类推。

## Returns

如果 input_number 被连接，则为 1；如果输入没有被连接，则为 0。
