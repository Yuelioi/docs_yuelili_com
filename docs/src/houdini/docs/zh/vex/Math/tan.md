`float tan(float n)`

返回`n`的三角正切值，其中`n`的单位是弧度。

`vector2 tan(vector2 v)`

`vector tan(vector v)`

`vector4 tan(vector4 v)`

返回一个新的向量，`tan()`应用于每个分量。
