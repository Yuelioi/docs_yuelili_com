`vector filamentsample(<geometry>geometry, vector position)`

在指定的 "位置 "对一组涡流丝定义的速度场进行采样。参数 "inputnum "或 "filename "指定从几何体中获取丝状曲线。强度 "和 "厚度 "的原始属性可用于几何体上，以定制每个长丝的强度和厚度。如果`inputnum`超出范围或`filename`无效，则返回零向量。
