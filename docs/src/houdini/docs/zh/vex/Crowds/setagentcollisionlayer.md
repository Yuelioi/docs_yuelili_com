警告

此函数已被弃用。请使用 [setagentcollisionlayers](setagentcollisionlayers.html) () ("设置代理基元的碰撞层。") 来代替。

## Arguments

`geohandle`

要写入的几何体的句柄。`geoself()`可以用来获取当前几何体的句柄。

`prim`

原始的数字。

`layername`

代理人的一个层的名称。
