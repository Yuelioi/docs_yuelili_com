`int setagentcurrentlayers(int geohandle, int prim, int layer_ids[])`

## Arguments

`geohandle`

Handle to the geometry to write to. `geoself()` can be used to get a handle to the current geometry.

`prim`

The primitive number.

`layernames`

A list of agent layer names.

`layer_ids`

A list of agent layer indices, as returned by [agentfindlayer](agentfindlayer.html) ("Finds the index of a layer in an agentâ€™s definition.").
