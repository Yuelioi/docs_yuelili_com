`int setprimgroup(int geohandle, string name, int prim_num, int value, string mode="set")`

## Arguments

`geohandle`

要写入的几何体的句柄。目前唯一有效的值是`0`或[geoself](geoself.html) () ("返回当前几何体的句柄。")，这意味着当前节点中的几何体。(这个参数将来可能会被用来允许写到其他的几何体)。

`name`

要修改的组的名称。

`prim_num`

要从组中添加或删除的原始编号。

`value`

`1'表示将基元放入组中，`0'表示将基元从组中移除。如果`mode`是`"toggle"`，这将被忽略。

`mode`

使用`"设置"`来根据`值`设置基元的成员资格。使用`"toggle"`来切换基元的成员资格，而不考虑`值'。
