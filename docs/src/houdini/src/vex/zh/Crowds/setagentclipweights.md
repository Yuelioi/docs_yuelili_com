`void setagentclipweights(int geohandle, int prim, float clipweights[])`

## Arguments

`geohandle`

要写入的几何体的句柄。`geoself()`可以用来获取当前几何体的句柄。

`prim`

原始的数字。

`clipweights`

砝码的清单。
