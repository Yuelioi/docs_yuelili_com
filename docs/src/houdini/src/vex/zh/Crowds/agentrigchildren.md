`int [] agentrigchildren(<geometry>geometry, int prim, int transform)`

返回给定变换的直接子节点的列表。

如果`transform`是[超出范围](agenttransformcount.html) () ("返回一个代理基元中的变换数量。")，`prim`超出范围，或者`prim`不是一个代理基元，则返回一个空数组。

## Arguments

`<geometry>`

当在一个节点的上下文中运行时（比如一个 wrangle SOP），这个参数可以是一个整数，代表要读取几何图形的输入数字（从 0 开始）。

或者，该参数可以是一个字符串，指定一个几何文件（例如，一个`.bgeo'）来读取。当在Houdini内部运行时，这可以是一个`op:/path/to/sop`的引用。

`prim`

原始的数字。

`transform`

代理人的装备中的一个变换的索引。

## Examples



访问一个给定变换的所有子女。

```c
int[] queue = { transform };

while (len(queue) > 0) {
int i = removeindex(queue, 0);
printf("%d\n", i);

foreach (int child; agentrigchildren(0, @primnum, i))
push(queue, child);
}

```
