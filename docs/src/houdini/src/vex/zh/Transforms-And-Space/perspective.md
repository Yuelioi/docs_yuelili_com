`matrix perspective(float zoom, float image_aspect, float pixel_aspect, float clip_near, float clip_far)`

`matrix perspective(float zoom, float image_aspect, float pixel_aspect, float clip_near, float clip_far, vector4 window)`

用给定的参数创建一个透视投影矩阵。这可以用来将点投射到摄像机的所谓 NDC 坐标上。

为了从世界空间到 NDC 空间进行一次转换，给定一个相机矩阵和一个投影矩阵，你可以使用，`worldToNDC = worldToCamera * projection;`。

## Arguments

`zoom`

镜头的变焦。有时变焦是用焦距和光圈来表示的。在这种情况下，`变焦=焦距/光圈`。

arg:image_aspect

图像的长宽比。有时 image_aspect 用`xres`和`yres`来表示。在这种情况下，`image_aspect = xres / yres'。

arg::clip_near

靠近剪接平面。

arg::clip_far

远处的剪接平面。

arg::window

window.x 和 window.y 是投影窗口的最小 xy 坐标，window.z 和 window.w 是投影窗口的最大 xy 坐标。这个参数是可选的，如果没有给出，则默认为{0,0,1,1}。
