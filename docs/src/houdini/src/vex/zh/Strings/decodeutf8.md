自 19.0 以来

`int [] decodeutf8(string str)`

将一个 UTF8 字符串转换为一系列的代码点。VEX 默认将其字符串视为 UTF8，但这意味着对字符串进行索引会有不寻常的行为。生成代码点可以确保每个逻辑字符都有一个索引。
