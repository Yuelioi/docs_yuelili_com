`vector curlnoise2d(float x, float y)`

`vector curlnoise2d(vector xyt)`

根据两个佩林噪声函数的导数的交叉乘积计算一个无发散矢量场。

所得的矢量都位于 X-Y 平面。

::: info Note

这与将 "卷曲噪声 "投射到一个平面上是不同的。

参见 VEX 语言指南中的[噪声和随机性](.../random.html)以获得更多信息。
