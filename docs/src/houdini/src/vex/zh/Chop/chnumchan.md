`int chnumchan()`

上下文中，"opinput "使用了"-1"。

`int chnumchan(int opinput)`

返回指定的输入中的通道数。

## Arguments

`opinput`

要读取的输入号码，从 0 开始。例如，第一个输入是 0，第二个输入是 1，以此类推。

如果你指定`-1`，函数使用当前的 CHOP 节点或输入`0`，如果它是连接的。
