`int hex_faceindex(int faceno, int vtxno)`

如果指定了一个无效的数字，则返回`-1'。

返回 "0 "到 "7"，指的是一般六面体的八个顶点。

## Arguments

`faceno`

六面体上的面。范围从`0`到`5`。

`vtxno`

返回四边形上的哪个顶点，`0`到`3`。从最低的数字开始，并遵循胡迪尼的绕线惯例。
