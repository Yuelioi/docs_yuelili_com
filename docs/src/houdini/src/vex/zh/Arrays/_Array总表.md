## 增

### append 追加值

Adds an item to an array or string.

向字符串尾部追加新的字符串

向数组尾部追加元素、其他数组

```c
void  append(string &array, string value)
```

```c
void  append(<type>&array[], <type>value)
```

```c
void  append(<type>&array[], <type>values[])
```
