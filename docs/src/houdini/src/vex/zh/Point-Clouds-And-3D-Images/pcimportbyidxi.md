`int pcimportbyidxi(int handle, string channel_name, int idx)`

在一个[pcopen](pcopen.html) ()（"返回一个点云文件的句柄。"）和一个[pcnumfound](pcnumfound.html)（"该节点返回 pcopen 发现的点的数量。"）之后，可以用来从发现的点中提取特定的搜索结果。

如果通道不存在，这将返回 0。
