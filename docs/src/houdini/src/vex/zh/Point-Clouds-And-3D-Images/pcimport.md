这个函数只在用`pciterate`或`pcunshaded`循环时有效。

`int pcimport(int handle, string channel_name, <type>&value)`

将点云文件中的数据导入给定的变量。

## Arguments

`channel_name`

有两个特殊的通道名称可以导入。

`point.number`

被处理的点的编号。

`point.distance`

被处理的点与查询点的距离。这只有在对无阴影的点进行迭代时才可用。

`value`

如果导入成功，该函数将用通道的值覆盖这个变量。

## Returns

`1`如果导入成功或`0`如果导入失败（通常是由于给定的通道名称不存在）。
