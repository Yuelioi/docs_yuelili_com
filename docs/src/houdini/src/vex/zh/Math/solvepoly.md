`int solvepoly(float coef[], float &roots[], int maxiter=0)`

## Arguments

`coef`

多项式的系数数组。

你必须对系数进行排序，使得`coef[i]`应该是`x^i`。**这与你正常写出多项式的顺序相反**。

`&roots`

该函数用多项式的实数根覆盖这个数组，以升序排列。

## Returns

实根的数量。
