`int setvertexpoint(int geohandle, int prim, int vtxofprim, int pt)`

将一个指定的顶点改成一个点号。

如果点号为-1，则不做任何重新布线。

如果 prim 是-1，则`vtxofprim`被视为线性索引，反之亦然。否则，一对(`prim`, `vtxofprim`)被用来识别基元顶点列表中的一个顶点。

这个函数是等效函数[setprimvertex](setprimvertex.html)（"将几何体中的一个顶点重新连接到不同的点。"）的新名称，是为了清晰起见而添加的。
