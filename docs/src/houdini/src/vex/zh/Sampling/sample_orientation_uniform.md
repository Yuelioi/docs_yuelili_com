`vector4 sample_orientation_uniform(vector u)`

## Arguments

`u`

0 和 1 之间的三个数字。

返回一个单位向量 4，即长度为 1 的向量 4，基于`u`。给出在`[0,1)`中三个值的均匀随机`u`向量，返回的单位向量将是均匀随机的，并且相对于`u`在单位超球的表面上是连续的。换句话说，它们将是均匀随机的定向四元数。
