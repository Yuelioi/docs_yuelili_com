`vector sample_direction_uniform(vector2 u)`

## Arguments

`u`

0 和 1 之间的一对数字。

返回一个单位向量，即长度为 1 的向量，基于`u`。给予`[0,1)`中的均匀随机的`u`对值，返回的单位向量将是均匀随机的，并且在单位球面上相对于`u`是连续的。
