`int sample_photon(light lp, vector &pos, vector &dir, float &scale, float time)`

Context(s) 从给定的光源产生一个光子，并返回场景中第一个交叉点的信息。`pos', `dir'和`scale'被填入光子在场景中的位置信息。

返回的整数代表反弹类型掩码（这是由光源上的照明标签类型决定的）。

如果光子没有与任何几何体相交，该函数返回 0。
