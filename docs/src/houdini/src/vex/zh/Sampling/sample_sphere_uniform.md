`vector sample_sphere_uniform(vector u)`

## Arguments

`u`

0 和 1 之间的三个数字。

返回一个长度<1 的向量，基于`u`。给出在`[0,1)`中三个值的均匀随机`u`向量，返回的向量将是均匀随机的，并且相对于`u`在单位球体中是连续的。
