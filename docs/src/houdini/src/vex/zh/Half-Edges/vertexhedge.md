`int vertexhedge(<geometry>geometry, int vertex)`

## Arguments

`<geometry>`

当在一个节点的上下文中运行时（比如一个 wrangle SOP），这个参数可以是一个整数，代表要读取几何图形的输入数字（从 0 开始）。

或者，该参数可以是一个字符串，指定一个几何文件（例如，一个`.bgeo'）来读取。当在Houdini内部运行时，这可以是一个`op:/path/to/sop`的引用。

`vertex`

几何图形中的线性顶点编号。`0`是第一个顶点。

## Returns

以`vertex`为源，以`vertex`的基元中`vertex`之后的顶点为目的的半边的编号。如果找不到相应的顶点，则返回`-1'。

## Examples



```c
int vtxhedge;

// Get the hedge out of vertex vertex number 2.
vtxhedge = vertexhedge("defgeo.bgeo", 2);

```
