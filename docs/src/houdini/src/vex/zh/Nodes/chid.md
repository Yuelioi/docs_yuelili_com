解决一个由通道路径或操作者路径给出的通道，并返回

其对应的运算符 id、参数 id 和 vector*index 通过输出参数。失败时返回-1 值。你也可以使用最后一个不接受通道*路径的重载函数来测试键的有效性。如果 id 是有效的，返回 1，否则返回 0。
