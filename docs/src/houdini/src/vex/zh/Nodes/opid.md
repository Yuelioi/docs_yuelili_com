`int opid(string op_path)`

`int opid(int op_id)`

解析一个操作者路径，并返回其相应的操作者 ID。

失败时返回-1。将操作员 ID 作为输入的表单，如果是有效的操作员 ID，则返回 1，否则返回 0，这可以用来测试操作员 ID 是否有效。
