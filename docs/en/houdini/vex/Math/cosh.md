## Description

`float cosh(float n)`

Returns the hyperbolic cosine of the argument.

`vector2 cosh(vector2 v)`

`vector cosh(vector v)`

`vector4 cosh(vector4 v)`

Returns a new vector with `cosh()` applied to each component.
