`int haspointattrib(<geometry>geometry, string attribute\_name)`

## Arguments

`<geometry>`

当在一个节点的上下文中运行时（比如一个 wrangle SOP），这个参数可以是一个整数，代表要读取几何图形的输入数字（从 0 开始）。

或者，该参数可以是一个字符串，指定一个几何文件（例如，`.bgeo'），以便从中读取。当在Houdini内部运行时，这可以是一个`op:/path/to/sop`的引用。

## Returns

如果该属性存在，则为 "1"，否则为 "0"。

## Examples



```c
// Determine if the P attribute exists.
int exists = haspointattrib("defgeo.bgeo", "P");

```
