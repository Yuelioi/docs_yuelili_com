`int has\_udim(string path)`

扫描输入路径中的特殊转换字符，以执行 UDIM 或 UVTILE 风格的文件名扩展。如果字符串有任何 UDIM/UVTILE 扩展模式，该函数返回 1，否则函数返回 0。
